This roadmap outlines the development phases of the self-hosted Nuxt Studio module. We're committed to build a powerful open-source content management system for Nuxt Content websites.

::prose-note
This roadmap is subject to change based on community feedback and evolving priorities. Join the conversation on [GitHub Discussions](https://github.com/nuxt-content/studio/discussions) or [join the Discord](https://discord.gg/sBXDm6e8SP) to help shape the future of Nuxt Studio.
::

## Current Status: Beta Release

::badge
v1.0.0-beta - Current Release
::

The beta release focuses on core functionality and stability. The beta now includes the TipTap visual editor (default), the Monaco code editor, and the new form-based editor for YAML/JSON files and frontmatter, providing a complete editing experience for all content types with MDC component support.

Support for GitLab as a git provider alongside GitHub.

Local development support without authentication.
