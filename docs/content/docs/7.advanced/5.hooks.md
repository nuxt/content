## `content:parsed`{lang="ts"}

This hook is called after the content is parsed and before it is saved to the database.

```ts
export default defineNuxtConfig({
  hooks: {
    'content:parsed'(ctx) {
      // { content: CollectionItemBase, collection: ResolvedCollection }
    }
  }
})
```
