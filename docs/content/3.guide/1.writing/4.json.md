# JSON

Nuxt Content can query JSON formatted content. The output contains all the document properties at its root.

## Example

::code-group

```json [content/hello.json]
{
  "title": "Hello Content v2!",
  "description": "The writing experience for Nuxt 3",
  "category": "announcement"
}
```

```js [Output]
{
  _path: '/hello',
  _draft: false,
  _partial: false,
  title: 'Hello Content v2!',
  description: 'The writing experience for Nuxt 3',
  category: 'announcement',
  _id: 'content:hello.json',
  _type: 'json',
  _source: 'content',
  _file: 'hello.json',
  _extension: 'json'
}
```

::
